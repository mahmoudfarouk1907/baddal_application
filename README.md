## 📖 About

Baddal App is a modern Flutter marketplace application that allows users to exchange products securely and efficiently.

The application provides authentication, product management, image uploads, and a clean user experience powered by Firebase services.

---

# ✨ Features

- 🔐 Firebase Authentication
- 📦 Product Exchange
- ❤️ Favorites
- 🔍 Search Products
- 🖼 Image Upload
- 👤 User Profiles

- 📱 Responsive UI
